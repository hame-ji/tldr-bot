1) AGENTS.md: Standardizing Instructions for AI Coding Agents

2)
*   **Purpose & Complement:** AGENTS.md is a Markdown file acting as a "README for agents," providing detailed technical context and actionable instructions specifically for AI coding agents, complementing human-focused `README.md`.
*   **Standardized Format & Location:** It is an open format, typically placed at the repository root (or subproject roots for monorepos), ensuring a predictable location for AI tools to find project-specific guidance.
*   **Content Guidelines:** Essential sections include Project Overview, Setup Commands, Development Workflow, Testing Instructions, Code Style Guidelines, and Build/Deployment details, all focused on providing executable commands and processes.
*   **Monorepo Support:** The format accommodates complex monorepo structures by allowing main `AGENTS.md` files at the root and nested files in subprojects, with the closest file taking precedence.
*   **Actionability & Best Practices:** Emphasizes providing specific, actionable commands within code blocks, maintaining currency, and verifying that all listed instructions work correctly.
*   **Ecosystem Compatibility:** AGENTS.md is designed to work across a wide range of AI coding tools and agents (20+), facilitating broad adoption.

3) Why it matters:
*   **Enables AI Automation:** Provides structured, actionable instructions that allow AI coding agents to understand, interact with, and contribute to codebases more effectively and autonomously.
*   **Improves Developer Workflow:** Reduces manual overhead and potential errors when integrating AI tools into development by giving them direct, precise guidance on project specifics.
*   **Standardizes AI Interaction:** Creates a common, predictable interface for various AI coding tools to consume project-specific information, enhancing compatibility and reliability across the AI-assisted development ecosystem.
