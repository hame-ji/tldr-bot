Title: Qwen3.6-Plus Released with Enhanced Agentic Coding

TL;DR: Alibaba Cloud has launched Qwen3.6-Plus, a highly capable multimodal model featuring a 1M token context window and significant upgrades to its agentic coding abilities.

Key points:
- The model defaults to a 1M context window and introduces a new `preserve_thinking` API feature to maintain reasoning context across multi-turn tasks.
- It excels at repository-level problem solving and can generate frontend code directly from visual inputs like UI screenshots or mockups.
- The API supports both OpenAI and Anthropic protocols, enabling seamless integration with tools like OpenClaw, Claude Code, and Qwen Code.

Why it matters:
- By combining deep reasoning, extensive memory, and precise tool execution, Qwen3.6-Plus offers software engineers a powerful, drop-in solution for building autonomous workflows and assisting with complex development tasks.

Evidence:
- The model scores 80.9 on SWE-bench Verified and 1517.9 on QwenWebBench, outperforming or matching frontier models like Claude Opus 4.5.

Caveat:
- All performance benchmarks and capability claims are self-reported by the model's creators, indicating potential evaluation bias.
