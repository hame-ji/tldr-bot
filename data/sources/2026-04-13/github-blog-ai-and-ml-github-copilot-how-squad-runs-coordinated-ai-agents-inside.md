Title: Squad for Repository-Based AI Coordination

TL;DR: Squad enables seamless, collaborative AI agent workflows directly inside GitHub repositories.

Key points:
- Simplifies multi-agent development with GitHub Copilot integration.
- Uses a "drop-box" pattern for persistent, versioned memory.
- Supports context replication across agents without large context splits.
- Makes it easy for engineers to start agentic workflows without heavy setup.

Why it matters:
- Reduces complexity in managing orchestration and infrastructure.
- Enhances collaboration and transparency in AI-driven development.
- Provides a scalable solution for team-based AI projects.
